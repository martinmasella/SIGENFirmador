# SIGENFirmador
Utilidad para tratamiento de archivos PDF

## 📄 Descripción

Este utilitario permite manipular archivos PDF utilizando las librerías iText y Tesseract.
Puede:
-Firmar digitalmente \n
-Empaquetar archivos dentro de un PDF \n
-Fusionar PDFs
-Dividir PDFs
-Ejecutar OCR
